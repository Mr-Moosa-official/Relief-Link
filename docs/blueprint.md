# **App Name**: ReliefLink

## Core Features:

- Incident Reporting: Allow users to submit real-time incident reports with location data via map integration.
- Volunteer Registration: Enable volunteers to register and create profiles with their skills and availability.
- Resource Tracking: Track available resources and donations with a tool that uses AI to determine if the new report suggests that a needed resource is now available. If it identifies that a needed resource is available, it will automatically mark the resource as found.
- Team Messaging: Provide a secure messaging platform for coordinated communication between relief teams.

## Style Guidelines:

- Primary color: Dark blue (#1A237E) to provide a sense of stability and trust.
- Background color: Dark gray (#37474F) to ensure readability and reduce eye strain.
- Accent color: Muted orange (#FF7043) for alerts and important information to draw attention effectively.
- Body and headline font: 'Inter' sans-serif for clear and efficient readability.
- Use clear, recognizable icons to represent different types of incidents and resources.
- Implement a clean, intuitive layout that prioritizes critical information and ease of navigation.
- Use subtle animations for loading states and updates to provide feedback without distracting from the primary task.
